# Pokemon Card Tracker

Mobile-first Pokemon card collection tracker built with Expo, React Native, Supabase, and the Pokemon TCG API.

## MVP

- Search Pokemon cards by name
- View card details and market price
- Save cards to your collection
- Track quantity, condition, purchase price, and notes
- Capture front and back photos of your real cards
- Store collection data in Supabase

## Getting started

```bash
npm install
cp .env.example .env
npm run start
```

Run `supabase/schema.sql` in your Supabase SQL editor before using the app.
